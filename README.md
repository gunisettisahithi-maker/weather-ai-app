# 🌦️ Weather App with AI Suggestions

This is a simple weather app built using Streamlit and Python.  
It shows real-time weather data and gives basic AI-based suggestions.

---

## Features

- Get weather details for any city 🌍  
- Shows temperature and humidity 🌡️  
- Displays simple weather messages  
- AI suggestions for daily use 🤖  

---

## Tech Used

- Python  
- Streamlit  
- OpenWeather API  
- Gemini AI  

---

## How to Run

1. Install requirements:
pip install -r requirements.txt

2. Run the app:
streamlit run app.py

---

## Note

Add your own API keys in the code before running.
